**About**

This is Herbivorene a NextJS website. Uses both React and Typescript. Uses shadcn/ui for UI components, and SQLITE for database. 

<br>

**Packages**
```
npm install
```

<br>

**Deploy** 

Development
``` 
npm run dev
``` 

Production
```
npm run build 
npm run start
```

<br>

Open [http://localhost:3000](http://localhost:3000)
